# Security Policy

This is a personal project maintained by Kyle Mason. If you believe you have found a security issue (for example an exposed credential, a sensitive value committed by mistake, or a vulnerability in published code), please report it privately rather than opening a public issue.

## Reporting

Email **masonkr@gmail.com** with details and, where possible, steps to reproduce. I will acknowledge receipt and address valid reports as quickly as I am able.

## Scope

These repositories document and support a personal home lab. Internal IP addresses and topology are intentionally published in some documents; those are not considered vulnerabilities. Reports of genuinely sensitive material (live credentials, private keys, tokens) are always welcome.
